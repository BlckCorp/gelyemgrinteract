# EC7 Workshop MVP

Интерактивный веб-справочник по **Geely Emgrand EC7**. Первый MVP сфокусирован на **JL4G18 1.8 MT** и разделе **ГРМ**.

## Что уже есть

- поиск по процедурам, артикулам и моментам затяжки;
- разделы руководства по ремонту;
- карточки процедур;
- таблица артикулов;
- список моментов затяжки;
- интерактивные схемы с кликабельными зонами;
- калькулятор ТО по пробегу с чек-листами;
- диагностика по симптомам с переходом в связанные разделы;
- изображения нужных страниц из руководства/каталога.

## Важно

Полные PDF-руководства не загружены в публичный репозиторий. В приложение перенесены только данные MVP и изображения нужных страниц. Перед реальным ремонтом нужно сверять данные с оригинальным руководством и соблюдать технику безопасности.

## Запуск

Открой `index.html` в браузере или включи GitHub Pages для ветки `main`, папка `/root`.

## Как расширять

1. Добавлять новые процедуры в `index.html` → массив `procedures`.
2. Добавлять новые запчасти в `index.html` → массив `parts`.
3. Добавлять моменты затяжки в `index.html` → массив `torques`.
4. Добавлять операции ТО в массив `maintenance`.
5. Добавлять подсказки диагностики в массив `symptoms`.
6. Для интерактивных схем добавлять элементы в `diagrams.items`, координаты указываются в процентах.
